# Pizza Website - React Bootstrap Project

Một website bán pizza được xây dựng bằng React và Bootstrap 5.

## Tính năng
- Carousel banner tự động chuyển ảnh
- Danh sách pizza dạng grid responsive
- Thẻ pizza với thông tin đầy đủ
- Hệ thống tag "Sale" và "New"
- Footer với thông tin cá nhân
- Responsive trên mọi thiết bị

## Cài đặt
```bash
npm install
npm start
